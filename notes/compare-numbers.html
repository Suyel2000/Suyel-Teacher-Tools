Compare Numbers

When we compare numbers, we find out which number is greater, smaller, or if the numbers are equal.

We use >, < or = to compare numbers.

What is Comparing Numbers?

  • Comparing numbers means finding out their relationship.
  • We can tell which number is greater, which is smaller, or if they are equal.
  • We compare numbers using their place value.
Teacher Tip: Always start comparing from the highest place value!

Quick Facts

  • If the first number is greater, use > (greater than).
  • If the first number is smaller, use < (less than).
  • If both numbers are the same, use = (equal to).
  • Compare digit by digit from the left (highest place).
  • If digits are the same, then compare the next place.
Greater Than
>

Means the number on the left is greater

8,745 > 6,321
Less Than
<

Means the number on the left is smaller

4,230 < 9,876
Equal To
=

Means both numbers are the same

5,678 = 5,678

How To Compare?

  1. Compare digits in the highest place value (Ten Thousands, if present).
  2. If the digits are equal, compare the next place value down.
  3. Continue till you find a greater or smaller digit.
  4. Use >, <, or = to compare.
Example: 45,672 and 45,167
Thousands: 4 = 4 | Hundreds: 6 > 1
So, 45,672 > 45,167

Examples Breakdown

23,456 > 21,987
2=2 (TTh), 3>1 (Th). So, 23,456 is greater.
18,230 < 18,780
1=1 (TTh), 8=8 (Th), 2<7 (H). So, 18,230 is smaller.
9,876 = 9,876
All digits match perfectly. Both are equal.
54,321 > 45,321
5>4 (TTh). So, 54,321 is greater.
67,890 < 67,980
6=6 (TTh), 7=7 (Th), 8<9 (H). So, 67,890 is smaller.
1,23,456 = 1,23,456
All digits match perfectly. Both are equal.
89,012 > 89,002
8=8 (TTh), 9=9 (Th), 0=0 (H), 1>0 (T). So, 89,012 is greater.
35,784 < 35,789
3=3 (TTh), 5=5 (Th), 7=7 (H), 8=8 (T), 4<9 (O). So, 35,784 is smaller.
