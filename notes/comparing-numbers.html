⭐ COMPARING NUMBERS ⭐

We compare numbers to know which is greater, smaller or equal.
WHAT DOES IT MEAN?

When we compare two numbers, we see which number is:

  • Greater ( > ) — means bigger
  • Smaller ( < ) — means less
  • Equal ( = ) — means the same
We compare numbers digit by digit from left to right. (From the highest place value).
SYMBOLS WE USE
Greater Than
>
Example:
52 > 36
(52 is greater than 36)
Smaller Than
<
Example:
18 < 25
(18 is smaller than 25)
Equal To
=
Example:
77 = 77
(77 is equal to 77)
🐊 CROCODILE RULE (ALLIGATOR METHOD)

The hungry crocodile's mouth always opens to eat the BIGGER number.

Greater Than ( > )

Mouth opens to the left because bigger number is on the left.

9 > 2

Smaller Than ( < )

Mouth opens to the right because bigger number is on the right.

3 < 7

Equal To ( = )

Numbers are equal. Crocodile is happy and takes a nap!

4 = 4
💡 QUICK FACTS
  • ★ Always compare from the leftmost digit (highest place value).
  • ★ If the digits are the same, move to the next place value.
  • ★ The number with more digits is always greater.
  • ★ If all digits are the same, the numbers are equal.

HOW TO COMPARE NUMBERS?
1Compare the digits in the highest place value.
2If the digits are different, the number with the greater digit is greater.
Example: Compare 372 and 298
Hundreds: 3 > 2, so 372 > 298
(We don't need to check tens and ones)
3If the digits are the same, move to the next place value and compare.
Example: Compare 428 and 421
Hundreds: 4 = 4 (same)
Tens: 2 = 2 (same)
Ones: 8 > 1, so 428 > 421
💡 TIPS TO REMEMBER
  • ✔ Look at the highest place value first.
  • ✔ If not same, decide immediately.
  • ✔ If same, move to the next place value.
  • ✔ More digits = Greater number.
    Example: 345 > 98
  • ✔ Same number of digits? Compare from left to right.
