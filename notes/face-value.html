FACE VALUE

The Value of a Digit Itself!
WHAT IS FACE VALUE?
  • The face value of a digit is the value of the digit itself.
  • It does not depend on its position in a number.
  • The face value of any digit is the digit itself.
No matter where the digit is, its face value is the same!
EXAMPLE

Look at the number

7
3
8
4
Digit 7 3 8 4
Face Value 7 3 8 4

So, the face value of 7 is 7, face value of 3 is 3, face value of 8 is 8 and face value of 4 is 4.

💡 QUICK FACTS
  • Face value of 0 is 0.
  • Face value of 1 is 1.
  • Face value of 2 is 2.
  • Face value of 9 is 9.
  • Face value is the same for all digits (0 to 9).
  • It does not change in any number.
FACE VALUE vs PLACE VALUE
Number Digit Face Value Place Value
5 3 2 5 5 500
3 3 30
2 2 2
The place value changes with the position, but the face value remains the same.
RULES TO REMEMBER
1
The face value of a digit is the digit itself.
Face Value of 6 = 6
2
Face value does not depend on its position.
In 6421, face value of 4 is 4.
In 1047, face value of 4 is also 4.
3
Every digit has only one face value.
Face value of 9 is always 9.
4
Face value is the same in all numbers.
Face value of 2 is 2 in 2, 20, 234, 9200 ...
MORE EXAMPLES
1
In the number 1 6 9
Face value of 1 = 1, 6 = 6, 9 = 9
2
In the number 5 0 7 2
Face value of 5 = 5, 0 = 0, 7 = 7, 2 = 2
3
In the number 9 8 0 1 6
Face value of 9 = 9, 8 = 8, 0 = 0, 1 = 1, 6 = 6
REAL LIFE EXAMPLES
🕒
In a clock, the face value of 3 is always 3, no matter where it is written.
🏷️
In price tags: ₹25, ₹52, ₹125 – face value of 2 is always 2.
🏠
In house numbers: 207, 720, 702 – face value of 0 is 0.
TIP TO REMEMBER
🧠💡
Face value is what the digit "faces" you with!
