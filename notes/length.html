MEASURING LENGTH

Length is how long something is from one end to another.
We use standard units to measure length accurately! 📐
WHAT IS LENGTH?
  • Length tells us how long something is from one end to the other.
  • We can measure length using non-standard and standard units.
  • Standard units help us measure accurately and compare easily.
Measure carefully, get it just right!

QUICK FACTS
  • We measure length in a straight line.
  • The standard unit for length is metre (m).
  • Smaller units are centimetre (cm) and millimetre (mm).
  • Bigger distance unit is kilometre (km).
  • Always align the start point exactly at 0 on a ruler.
  • Choose the right tool for accurate measurement.
Always compare units of the same kind!

HOW TO MEASURE LENGTH USING A RULER
1 Place the Object
Lay the object flat along the edge of the ruler.
2 Start at Zero (0)
Make sure one end of the object starts exactly at the 0 mark.
3 Read the Number
Look at the other end. The number it matches is the total length!

CONVERSION EXAMPLES
1. Convert cm to mm: 5 cm = 5 × 10 = mm
2. Convert m to cm: 3 m = 3 × 100 = cm
3. Convert km to m: 2 km = 2 × 1000 = m
4. Complex Step: 150 cm = 1 m + cm
