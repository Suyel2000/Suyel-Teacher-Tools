A pattern is a rule. Numbers follow the rule in a certain order.
1. WHAT IS A NUMBER PATTERN?

A number pattern is a series of numbers that follow a rule.

Example:

Rule: Add 2 each time.
This is a pattern.

3. HOW TO FIND A PATTERN?
  1. Look at the numbers carefully.
  2. Find the difference between one number and the next.
  3. See if the same difference is added or subtracted.
  4. Write the rule.

Example:

Rule: Add 4 each time.

Next number: 19 + 4 = 23
