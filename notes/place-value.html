PLACE VALUE
Understand it. Use it. Excel in Maths!
Place value tells us the value of a digit based on its position in a number.
From right: Ones → Tens → Hundreds → Thousands → Ten Thousands → Lakhs → Ten Lakhs → Crores (1 Crore)
WHAT IS PLACE VALUE?
  • Every digit in a number has a value depending on where it is placed.
  • The same digit can have different values at different positions.
  • It helps us read, write, compare and understand numbers better.
"Same digit, different place, different value!"
QUICK FACTS (UP TO 1 CRORE)
  • Ones (O) = 1
  • Tens (T) = 10
  • Hundreds (H) = 100
  • Thousands (Th) = 1,000
  • Ten Thousands (TTh) = 10,000
  • Lakhs (L) = 1,00,00,000
  • Ten Lakhs (TL) = 10,00,000
  • Crores (C) = 1,00,00,000 (1 Crore)
  • Indian Comma System: First comma after 3 digits from right, then after every 2 digits! (e.g., 1,00,00,000)

1 POSITION MATTERS

The value of a digit changes dynamically based on its position in a 4-digit number sequence.

Example: In 7,625
• 7 in thousands place = 7,000
• 6 in hundreds place = 600
• 2 in tens place = 20
• 5 in ones place = 5
2 HOW TO PUT COMMAS

In the Indian system, commas separate periods to make big numbers up to 1 Crore easy to read:

The Rule:
1. First comma goes after 3 digits from the right.
2. Next commas go after every 2 digits.
Examples up to Crore:
• 10000000 → 1,00,00,000 (1 Crore)
• 756432 → 7,56,432
• 98765 → 98,765
