WHAT ARE ROMAN NUMERALS?

  • ★ Roman numerals are a system of expressing numbers using letters.
  • ★ They were used in Ancient Rome.
  • ★ We still see them today on clocks, book chapters, movie sequels, buildings, coins and more!
Let's learn this amazing number system! 📚

3. IMPORTANT POINTS

  • ★ A letter cannot be repeated more than 3 times. (IIII ❌, but IV = 4)
  • ★ Symbols are written from largest to smallest (Left to Right).
  • ★ Use subtraction rule only for the pairs shown below:
I before V (IV)
I before X (IX)
X before L (XL)
X before C (XC)
C before D (CD)
C before M (CM)
⚠️ No other subtraction allowed (IC ❌, IL ❌, VX ❌, XD ❌)
★ There is no symbol for zero (0).

QUICK FACTS 💡

  • ★ Roman numerals do not have a zero (0).
  • ★ They are written using combinations of 7 letters.
  • ★ The same letter cannot be written more than 3 times in a row.
  • ★ When a smaller value comes before a larger value, we subtract.
  • ★ When a smaller value comes after a larger value, we add.
Follow the rules and become an expert! ✨

2. RULES TO WRITE ROMAN NUMERALS

A. REPEATED ADDITION
(Smaller letter comes after larger letter)

Add the values:

I + I = II (1 + 1 = 2)
X + X = XX (10 + 10 = 20)
C + C + C = CCC (100 + 100 + 100 = 300)
M + M = MM (1000 + 1000 = 2000)

B. SUBTRACTION RULE
(Smaller letter comes before larger letter)

Subtract the smaller value from the larger value:

I before V = IV (5 - 1 = 4)
I before X = IX (10 - 1 = 9)
X before L = XL (50 - 10 = 40)
X before C = XC (100 - 10 = 90)
C before D = CD (500 - 100 = 400)
C before M = CM (1000 - 1000 = 900)

6. HOW TO CONVERT

A. HINDU-ARABIC TO ROMAN

  1. Write the number.
  2. Break it into thousands, hundreds, tens, and ones.
  3. Write the Roman numeral for each part.
  4. Combine from largest to smallest.
Example: 1987
1987 = 1000 + 900 + 80 + 7
     = M + CM + LXXX + VII
     = MCMLXXXVII

B. ROMAN TO HINDU-ARABIC

  1. Read the numeral from left to right.
  2. If a smaller value comes before a larger one, subtract it.
  3. If a smaller value comes after a larger one, add it.
  4. Placeholder step for perfect alignment.
Example: MCMXLIV
= M (1000) + CM (900) + XL (40) + IV (4)
= 1000 + 900 + 40 + 4
= 1944
