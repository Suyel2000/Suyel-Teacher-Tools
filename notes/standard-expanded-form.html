STANDARD AND EXPANDED FORM

⭐ Two ways to show a number! ⭐
WHAT ARE THEY?
STANDARD FORM:

The normal way of writing a number using digits.

EXPANDED FORM:

The way of writing a number as the sum of the values of each digit based on its place.

💬 Expanded form helps us see the value of each digit!
PLACE VALUE REMINDER
Thousands Hundreds Tens Ones
Th H T O
1000 100 10 1

The value of a digit depends on its place (position).

Example: In 348, 3 is in hundreds place, so its value is 300. 💡
QUICK FACTS 💡
  • In standard form, we write the number in the usual way.
  • In expanded form, we add the value of each digit.
  • Expanded form shows the place value of each digit clearly.
  • A number can have both forms.

TIPS TO REMEMBER
  • Look at the place of each digit.
  • Multiply the digit by its place value.
  • Then add all the values to write expanded form.
  • In expanded form, write each value from largest place to smallest place.
