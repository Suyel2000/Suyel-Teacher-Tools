SUBTRACTION

Take Away • Find the Difference • Solve with Confidence!

Subtraction helps us find how many are left, how many more, or the difference between numbers.

WHAT IS SUBTRACTION?

  • Subtraction means
    "taking away one number from another".
  • We use the minus sign (−) to subtract numbers.
  • It is one of the four basic operations in mathematics.
Let's take away and make math easy and fun!

QUICK FACTS

  • 0 − any number = Not possible (for whole numbers)
  • Any number − 0 = That number
  • If the top number is smaller, we regroup (borrow).
  • Subtraction is not commutative.
    (a − b ≠ b − a)
  • Practice subtraction to become fast and accurate!
TYPES OF SUBTRACTION
1 SUBTRACTION WITHOUT REGROUPING (No Borrowing)
When each digit in the top number is greater than or equal to the digit below it.
2 SUBTRACTION WITH REGROUPING (With Borrowing)
When a digit in the top number is smaller than the digit below it.
HOW TO SUBTRACT WITHOUT REGROUPING
  1. Write numbers in columns (place value aligned).
  2. Start subtracting from the ones place.
  3. Then tens, hundreds...
  4. Write the difference.
Example:
HTO 432 −210
222
Step-by-step:
Ones: 2 − 0 = 2
Tens: 3 − 1 = 2
Hundreds: 4 − 2 = 2

So, 432 − 210 = 222
TIP: Always line up the digits according to place value.
HOW TO SUBTRACT WITH REGROUPING
  1. Write numbers in columns.
  2. If a digit in the top row is smaller, borrow 1 from the next place value on the left.
  3. Subtract.
  4. Continue for all places.
Example:
HTO 3912 402 −158
244
Step-by-step:
Ones: 12 − 8 = 4 (We borrow 1 ten)
Tens: 9 − 5 = 4 (We borrowed 1 hundred)
Hundreds: 3 − 1 = 2
So, 402 − 158 = 244
TIP: Borrow (regroup) from the next place value on the left.
SUBTRACTION WITHOUT REGROUPING (NO BORROWING)
2-DIGIT SUBTRACTION

So, 72 − 35 = 37

TIPS TO REMEMBER
  • Always write numbers in columns.
  • Line up the digits by place value.
  • If the top digit is smaller, regroup (borrow).
  • Don't forget to subtract the borrowed number in the next column.
  • Check your answer by adding back!
